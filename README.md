# MediaPy

- Python >= 3.11
- pip

## Ejecutar programa en modo dev

1. Crear VENV

```console
python3 -m venv venv
```

2. Correr requirements

```console
python -m pip install -r requirements.txt
```

3. Ejecutar

```console
python main.py
```

3. Ya puedes utilizar el archivo [main.exe](output/main.exe)
